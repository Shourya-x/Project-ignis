System Overview

🎯

Precision Control

Real-time attitude control with complementary filter and PID algorithm for stable flight.

📡

LoRa Telemetry

Long-range telemetry up to 5km with SX1278 module for real-time data monitoring.

🛰️

GPS Tracking

NEO-6M GPS module for position tracking, speed, and altitude data.

⚙️

Modular Design

Separate modules for sensor hub, TVC mechanism, and ground station.
